## 各阶段详情

### ✅ P1：基础刷题框架
- 项目脚手架（React 18 + Vite 6.3 + Tailwind CSS v4）
- 全局类型系统（5 种题目联合类型）
- 数据自动扫描（`import.meta.glob`）
- 学科 → 章节 → 题目三级导航
- 练习流程（逐题作答 + 即时反馈）
- 五类题目组件（选择 / 填空 / 计算 / 判断 / 证明）
- KaTeX 公式渲染 + 化学公式

### ✅ P2：错题本 + 统计
- 自动收录错题 + 手动移除
- 按学科筛选错题
- 刷题进度 + 正确率 + 错题率统计
- 进度条 / 难度星级 / 标签徽章 UI 组件

### ✅ P3：模拟考试
- 智能组卷（Fisher-Yates 洗牌 + 难度配比）
- 考试流程（倒计时 + 答题卡 + 自动交卷）
- 成绩报告（分数 + 分区得分 + 错题回顾）

### ✅ P4：TTS 语音朗读
- Web Speech API 封装
- LaTeX → 中文口语转换（100+ 映射规则）
- 题目卡片 TTS 按钮集成

### ✅ P5：B站视频嵌入
- B站 iframe 播放器 + IntersectionObserver 懒加载
- 时间区间播放（计时器 + 覆盖层遮罩）
- 片段播放器（多片段选择 + 联动跳转）
- 解析步骤级视频按钮

### ✅ P5.5：几何构造引擎
- 构造指令 DSL：25 种操作类型（定点、极坐标、中点、垂足、交点、外心、内心、垂心等）
- 几何引擎核心：纯数学计算，确定性公式算坐标，无 AI 猜测（`src/utils/geoEngine.ts`，571 行）
- JSXGraph 渲染链路：GeometryBoard + figureBuilders（点、线、圆、角标记、刻度线、标签）
- 函数图像画板：二次函数 / 一次函数绘图
- 已验证场景：三角形、平行四边形、矩形、正方形、梯形、圆、切线、外接圆
- 设计文档：`docs/geo-engine-plan.md`

### ⬜ P6：题库扩充
- 数学：扩充至每章 15-20 题，覆盖 7-9 年级全章节
- 物理：扩充至每章 15-20 题，覆盖 8-9 年级
- 化学：扩充至每章 15-20 题，覆盖 9 年级
- 目标：总计 200+ 道题
- 加入中考真题标签和来源信息

### ⬜ P7：PWA 离线支持
- Service Worker 缓存静态资源
- Web App Manifest（安装到桌面）
- 离线答题 + 上线同步

### ⬜ P8：部署上线
- Vercel / Netlify 部署配置
- SPA 路由回退
- 域名 + HTTPS
